JS是什么

JavaScript（JS）是一种高级、解释性的编程语言，主要用于在网页上实现交互性和动态效果。它最初由Netscape公司开发，最初被称为“LiveScript”，后来为了借助Java的成功，改名为JavaScript。虽然名字中带有Java，但实际上两者在语法和设计上有很大的不同。

前置知识DOM

DOM（文档对象模型）是一种表示和操作HTML、XML等文档的标准化的接口。在网页上，DOM 将整个文档抽象成一个树状结构，通过这个结构可以方便地访问和修改文档的内容、结构和样式。

以下是 DOM 的一些基本概念和操作：

1. 节点（Node）:

DOM 树的基本构建块是节点。节点可以是元素、属性、文本等。常见的节点类型包括：

  • 元素节点（Element Node）： 表示 HTML 元素。
  • 属性节点（Attribute Node）： 表示 HTML 元素的属性。
  • 文本节点（Text Node）： 表示 HTML 元素中的文本内容。

2. DOM 树结构：

DOM 树的结构是层次化的，根节点是文档节点，文档节点包含整个 HTML 文档的内容。每个节点都可以有子节点和父节点，形成一个层次结构。

在这个例子中，<html> 是根节点，它有一个子节点 <head> 和一个子节点 <body>，而 <body> 又包含两个子节点 <h1> 和 <p>。

3. 获取元素：

通过 JavaScript 可以通过不同的方法获取元素，例如 getElementById、getElementsByClassName、getElementsByTagName 和 querySelector 等。

4. 操作元素：

可以通过 DOM 提供的方法和属性来操作元素，例如修改元素的文本内容、样式、添加或删除元素等。

5. 事件：

DOM 允许通过事件机制来响应用户的交互。通过 addEventListener 方法可以为元素添加事件监听器，响应特定的事件类型。

变量

在应用程序中，使用变量来作为值的符号名。变量的名字又叫做标识符，其需要遵守一定的规则。

一个 JavaScript 标识符必须以字母、下划线（_）或者美元符号（$）开头；后续的字符也可以是数字（0-9）。因为 JavaScript 语言是区分大小写的，所以字母可以是从“A”到“Z”的大写字母和从“a”到“z”的小写字母。

声明变量

你可以用以下三种方式声明变量：

使用关键词 var 。例如 var x = 42。这个语法可以用来声明局部变量和全局变量。

直接赋值。例如 x = 42。在函数外使用这种形式赋值，会产生一个全局变量。在严格模式下会产生错误。因此你不应该使用这种方式来声明变量。

使用关键词 let 。例如 let y = 13。这个语法可以用来声明块作用域的局部变量。參考下方变量的作用域 (Variable scope)。

你可以使用解构赋值将对象字面量的属性绑定到变量。就像这样 let { bar } = foo。这会创建一个名为 bar 的变量，并且将 foo 对象中属性名与之相同的属性的值赋给它。

你也可以直接给属性赋值。像这样 x = 42。这种声明方式将会创建一个未声明全局变量。这样做还会产生 JavaScript 警告。因为未声明的全局变量常常导致预期之外的行为，所以不建议使用。

事件基础

在前端开发中，事件是指用户在页面上执行的各种交互动作，如点击按钮、输入文本、鼠标移动等。JavaScript 提供了事件处理机制，使开发者能够捕获和响应这些事件。

1. 事件类型：

常见的事件类型包括：

  • 鼠标事件： click（点击）、mouseover（鼠标悬停）、mouseout（鼠标移出）等。
  • 键盘事件： keydown（键被按下）、keyup（键被释放）等。
  • 表单事件： submit（表单提交）、change（表单元素值改变）等。

2. 事件对象：

当事件发生时，浏览器会创建一个事件对象，该对象包含有关事件的信息，例如触发事件的元素、事件类型、鼠标位置等。事件处理函数的参数通常是事件对象。

3. 事件冒泡和捕获：

事件在 DOM 树中传播，有两个阶段：捕获阶段和冒泡阶段。通过 addEventListener 的第三个参数，可以指定是在捕获阶段处理事件还是在冒泡阶段处理事件（默认是冒泡阶段）。

4. 阻止默认行为和阻止冒泡：

  • event.preventDefault()： 阻止事件的默认行为，如阻止链接跳转、表单提交等。
  • event.stopPropagation()： 阻止事件的冒泡传递。

异步编程

异步编程是一种在程序执行期间允许其他操作继续进行的编程方式。在 JavaScript 中，由于某些操作（例如网络请求、文件读取、定时器等）可能需要较长时间才能完成，采用异步编程可以使程序在等待这些操作完成的同时继续执行其他任务，而不会阻塞整个程序

1. 回调函数：

回调函数是一种基本的异步编程模式，通过将函数作为参数传递给另一个函数，在异步操作完成后执行回调。

2. Promise：

Promise 是一种更现代、更可读的异步编程方式，它代表一个异步操作的最终完成或失败。Promise 对象有三种状态：pending（进行中）、fulfilled（已成功）和rejected（已失败）。

3. async/await：

async/await 是基于 Promise 的语法糖，提供了一种更清晰和更同步的写法，使异步代码看起来像同步代码。

4. 事件监听：

通过事件监听可以实现异步编程，例如处理用户输入、鼠标点击等事件。

这些是 JavaScript 中常见的异步编程模式和工具。选择哪种方式取决于具体的需求和个人或团队的编码风格。异步编程是现代 JavaScript 开发中不可或缺的一部分，特别是在处理网络请求和用户交互等方面。

对象

在编程中，对象是一种数据结构，它可以包含数据和代码（函数），并以键值对的形式组织。在不同的编程语言中，对象可能被称为类、结构体、字典等，但它们都共享类似的概念。

在 JavaScript 中，对象是一种非常重要的数据类型，它用于表示复杂的数据结构和实现面向对象编程。以下是 JavaScript 中对象的一些基本概念：

1. 对象的创建：

在 JavaScript 中，可以通过对象字面量、构造函数、Object.create 等方式来创建对象。

2. 访问对象的属性和方法：

可以使用点运算符或方括号运算符来访问对象的属性和方法。

3. 修改对象：

可以随时修改对象的属性和方法。

4. 对象的迭代：

可以使用 for...in 循环来迭代对象的属性。

5. 原型和继承：

JavaScript 中的对象通过原型链实现继承关系。每个对象都有一个原型（prototype），可以通过原型来共享属性和方法。

6. 内置对象：

JavaScript 提供了许多内置对象，如 Array、Date、RegExp 等，它们有预定义的属性和方法。

JavaScript 中的对象是一种灵活且强大的数据结构，它在构建应用程序时起到关键作用。对象的属性和方法可以用于组织和操作数据，实现面向对象编程的概念。
